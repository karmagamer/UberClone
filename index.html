Uber clone

Ride pick up mobile application for android. Similar to Uber. Implemented driver-side and passenger side of the application.

View project on GitHub

Welcome to Uber Android Application.

Goal of the project was to create android application for users and drivers. Application successfully allows user to request a ride and allows drivers to pickup passengers/users. Google maps api, Parse Server on Amazon EC-2 for real time updates.
